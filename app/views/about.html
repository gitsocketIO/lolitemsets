Select a map

You must select at least one of the three maps.

Your item set will be available only for the maps that you have chosen.

Select your champion

You just have to select the champion for whom you wish to create a set.

Create your item sets

The real challenge starts here ! Select the items that you want to use and organize it !

Graphic interface supports drag-n-drop files. Enjoy !

Download and Share

Your set is ready ? Download it !

Share your set with the community ! Then other players will be able to vote for it and to download it too !

Install your set

Your set will be download in a .json file.
You will need to put this file in the right place: League of Legends\Config\Champions\{championKey}\Recommended\

For example: C:\Riot Games\League of Legends\Config\Champions\Nidalee\Recommended
